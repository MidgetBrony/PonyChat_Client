PonyChat Client Beta 1.0

What is it

A specific Client Written for PonyChat users that only connect to the ponychat network that has features removed/added for Specifically Elemental-IRCD. The Orginal Source Code is based off IceChat 9 RC 8 20131027.

Supported Platforms

Its Mono/C# Based Just like the orginal IceChat, so if your OS can run Mono/.Net, it more then likely will run the Ponychat Client.
